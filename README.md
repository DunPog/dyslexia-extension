### Local development setup:
1. Clone repository into an IDE of choice
2. Open the terminal and run the command: ```npm install```
3. After packages have been installed, run the command: ```npm run dev```
4. Open Chrome/Your Chromium-based browser of choice and manage extensions (On Chrome it is ```chrome://extensions/```)
5. Enable Developer mode if required and click Load unpacked
6. Navigate to this project directory and load the ```dist``` folder
7. Enable the extension
